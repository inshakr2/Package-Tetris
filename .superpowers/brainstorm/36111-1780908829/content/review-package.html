접근안 상세: A안 5단계 기준

A안은 `공간 -> 블록 -> 검토 -> 결과 -> 추가`의 선형 플로우입니다. B안은 입력과 3D가 한 화면에 몰려 과밀하고, C안은 라이브러리 관리 중심이라 첫 계산까지 이동이 많습니다.

Step 1~3 상세: 입력 플로우

단계검토 가능한 상태CTA 조건

3 실행 전 검토실행 가능, 경고 포함 가능, 실행 불가, 실행 중, 실행 실패오류 없으면 `시뮬레이션 실행` 활성, 실행 중 `계산 중...`
